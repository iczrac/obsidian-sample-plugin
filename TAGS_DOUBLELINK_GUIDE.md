# 🏷️🔗 标签+双链功能完整指南

## 🎯 功能概述

我们已经成功集成了Obsidian的标签功能与双链功能，创造了一个强大的八字知识管理系统！

### ✨ 核心特色
- **智能标签生成** - 基于八字信息自动生成多维度标签
- **双链网络** - 自动建立人物、神煞、格局等关联
- **标签索引** - 创建分类索引页面便于查找
- **知识图谱** - 形成完整的命理知识网络

## 🚀 使用方法

### 第一步：创建带姓名的八字
```markdown
```bazi
date: 1990-01-01 08:00
gender: 男
name: 张三
```

### 第二步：使用双链工具栏
八字命盘上方会显示工具栏：
- 🔗 **显示相关链接** - 查看所有相关双链
- 📝 **创建相关笔记** - 一键生成完整知识库
- 👤 **打开人物档案** - 直接跳转个人主页

### 第三步：享受智能标签系统
点击📝按钮后，系统会自动创建：

#### 个人档案（张三.md）
```markdown
# 张三

## 基本信息
- 出生日期：1990-01-01
- 性别：男
- 生肖：马

## 八字信息
```bazi
date: 1990-01-01 08:00
gender: 男
name: 张三
```

## 智能标签
#人物档案 #八字 #男性 #马年 #90后 #庚日主 #五行金 #天乙贵人 #正财格
```

#### 标签索引页面
- **五行标签索引.md** - 按五行分类查找
- **神煞标签索引.md** - 按神煞类型分类
- **年代标签索引.md** - 按年代世代分类

## 🏷️ 智能标签系统

### 基础标签
- `#人物档案` - 标识为人物档案
- `#八字` - 八字相关内容
- `#男性` / `#女性` - 性别标识
- `#马年` - 生肖标识
- `#90后` - 世代标识
- `#1990年代` - 年代标识

## 🔍 高级搜索功能

### 标签组合搜索
在Obsidian搜索栏中使用：

- `tag:#金日主 tag:#天乙贵人` - 查找金日主且有天乙贵人的人
- `tag:#90后 tag:#正财格` - 查找90后正财格的人
- `tag:#男性 tag:#桃花` tag:#驿马` - 查找有桃花驿马的男性

### 双链追踪
- 点击 `[[天乙贵人详解]]` 查看神煞说明
- 点击 `[[1990年生人]]` 查看同年生人
- 点击 `[[正财格]]` 查看格局分析
